## 1. 系统概述

### 1.1 功能简介

本系统是一套完整的游泳比赛管理与计时解决方案，覆盖赛事全生命周期：

- **赛前**: 赛事创建、运动员报名、自动生成日程、预赛分组编排、纪录数据管理
- **赛中**: 实时计时控制、泳道设备管理、分段成绩显示、第1名成绩高亮、WR/CR纪录对比
- **赛后**: 成绩确认与排名、晋级管理、成绩发布、文档打印（秩序册/出发表/成绩册等）

### 1.2 系统组成

| 组件 | 说明 | 运行方式 |
|------|------|---------|
| **主服务器** (SwimmingScoreboard.exe) | 赛事管理核心，连接计时硬件 | Windows 应用程序 |
| **远程控制台** (RemoteTimingControl.exe) | 独立的比赛控制客户端 | Windows 应用程序 |
| **Web控制台** (race_control.html) | 浏览器端比赛控制界面 | 任意浏览器打开 |
| **网页检录台** (checkin.html) | 赛前检录、打印检录表 | 任意浏览器打开 |
| **大屏显示** (display.html) | 观众大屏幕显示 | 浏览器全屏 |
| **排名屏** (leaderboard.html) | 实时排名显示 | 浏览器全屏 |
| **在线报名** (register.html) | 运动员自助报名 | 浏览器打开 |

### 1.3 系统要求

- **操作系统**: Windows 7/8/10/11
- **运行环境**: .NET Framework 4.0（Windows 7+ 自带）
- **网络**: 局域网（主服务器与客户端/大屏需在同一网络）
- **端口**: 3002（WebSocket通信）
- **浏览器**: Chrome / Edge（推荐）

## 2. 安装与启动

### 2.1 安装

1. 解压安装包 `游泳赛事管理系统_安装包.zip`
2. 双击 **安装.bat**
3. 输入安装目录（直接回车使用默认路径 `C:\SwimmingTimingSystem`）
4. 安装完成后桌面出现快捷方式

### 2.2 启动主服务器

1. 双击桌面 **"游泳赛事管理主服务器"** 快捷方式
2. 首次启动需要设置管理员密码
3. 登录后进入主界面

### 2.3 启动远程控制台

1. 双击桌面 **"远程计时控制台"** 快捷方式
2. 在顶部输入主服务器的IP地址和端口（如 `192.168.1.100:3002`）
3. 点击"连接"

### 2.4 打开Web控制台

在浏览器地址栏输入：
```
http://主服务器IP:3002/race_control.html
```

### 2.5 防火墙设置

首次运行主服务器时，Windows 可能弹出防火墙提示。请选择 **"允许访问"**，确保端口 3002 可用。

## 4. 赛事管理与报名

主服务器的 **"赛事管理与报名"** 标签页包含以下子标签：

### 4.1 赛事概览

- 显示当前赛事基本信息（名称、日期、地点等）
- 统计已注册运动员数、项目数、团队数等概览数据
- 编辑赛事名称、举办单位、裁判信息等

### 4.2 运动员管理

- **添加运动员**: 填写姓名、性别、代表队、出生日期、身份证号等
- **导入CSV**: 批量导入运动员数据（格式：号码,姓名,性别,代表队,项目,报名成绩,...）
  - 支持 UTF-8 和 GBK 两种编码自动识别
- **参赛项目管理**: 为每位运动员分配参赛项目和报名成绩
- **在线报名**: 运动员可通过 `http://主服务器IP:3002/register.html` 自助报名

### 4.3 接力队管理

- 创建接力队（4人一队）
- 指定棒次顺序
- 设置接力项目（4×100米自由泳接力 等）

### 4.4 出场编排微调

- 在自动分组后，手动微调运动员的组次和泳道
- 拖拽操作或直接编辑数据

### 4.5 赛程管理

- **一键生成日程**: 根据报名数据自动生成完整赛程
  - 自动安排预赛→半决赛→决赛的时序（严格保证时间顺序）
  - 根据项目距离计算合理的时间间隔（50米项目5分钟/组，1500米项目22分钟/组）
  - 三阶段项目的预赛安排在前几天，保证半决赛和决赛有充足间隔
- **预赛自动分组**: 按报名成绩蛇形分配泳道
- **修改赛程**: 可手动调整日期、时间、组次

### 4.6 纪录管理

详见 [第7章 纪录管理](#7-纪录管理)

## 5. 比赛控制

### 5.1 界面布局

比赛控制界面分为三栏：

| 区域 | 内容 |
|------|------|
| **左栏** | 赛程导航树 + 计时源对比 |
| **中间** | WR/CR纪录 + 滚动时间 + 泳道实时状态 + 日志 |
| **右栏** | 比赛控制按钮 + 泳道操作 + 显示控制 |

### 5.2 比赛流程

#### 第一步：选择项目

在左栏赛程导航树中点击要比赛的组次（如"女 100米蝶泳 预赛 第1组"）。

> **注意**: 比赛进行中（就位/比赛中状态）不能切换项目，必须先"计时复位"。

#### 第二步：就位

点击 **"就位"** 按钮：
- 状态变为"就位"（蓝色指示灯）
- 泳道设备重置为初始状态
- 出发端出发台打开

#### 第三步：发令

点击 **"发令"** 按钮：
- 状态变为"比赛中"（绿色指示灯）
- 计时器开始（滚动时间显示）
- 所有触板和盲表关闭，泳道关闭倒计时开始
- 倒计时结束后触板/盲表自动打开，准备接收数据

#### 第四步：比赛进行中

比赛中系统自动处理：
- **泳道关闭/打开**: 按设定的倒计时自动控制
- **分段成绩**: 每次触板触碰显示当前分段累计时间，保持"分段成绩显示时长"设定的秒数后消隐
- **实时名次**: 与分段成绩同步显示/消隐，根据分段数和累计时间实时排序
- **反应时**: 出发台检测到反应时间后显示相同时长
- **第1名成绩**: 每个分段点最先触板的成绩在滚动时间位置以**绿色**显示，保持"第1名成绩停留时间"设定的秒数

#### 第五步：确认成绩

所有泳道完赛后，点击 **"确认本组成绩"**：
- 弹出确认对话框显示当前项目/组次信息
- 确认后成绩锁定保存
- 自动检查是否打破纪录

### 5.3 比赛控制按钮

| 按钮 | 功能 |
|------|------|
| **就  位** | 进入就位状态，重置泳道设备 |
| **发  令** | 开始比赛，启动计时 |
| **计时复位** | 重置计时器和设备；**DSQ/DNS/DNF状态保留** |
| **确认本组成绩** | 锁定保存当前组成绩 |

### 5.4 泳道操作

需要先在输入框中输入道次号码，再点击操作按钮：

| 按钮 | 功能 | 说明 |
|------|------|------|
| **DNS** | 缺席未出发 | 运动员未到场 |
| **DNF** | 中途退出 | 因伤病等原因中途退赛 |
| **DSQ** | 犯规取消资格 | 违反比赛规则 |
| **手动输入** | 手动输入成绩 | 在无自动计时数据时人工录入 |
| **全部打开** | 打开所有泳道设备 | 触板和盲表全部打开 |
| **全部关闭** | 关闭所有泳道设备 | 触板和盲表全部关闭 |

> DNS/DNF/DSQ 操作都有确认对话框防止误操作。
> 手动输入成绩有**二次确认**，确认后直接写入数据库。
> 计时复位时 DSQ/DNS/DNF 状态**不会被清除**（按规则这些运动员不能再参赛）。

### 5.5 显示控制

控制大屏幕上显示的内容：

| 按钮 | 功能 |
|------|------|
| **出发表** | 显示当前组的出发名单 |
| **比赛视图** | 显示实时比赛画面 |
| **组成绩** | 显示当前组的比赛成绩 |
| **总排名** | 显示项目总排名 |
| **团  体** | 显示团体积分排名 |
| **颁  奖** | 显示颁奖信息 |
| **纪  录** | 显示纪录信息 |
| **欢  迎** | 显示欢迎画面 |
| **成绩发布** | 发布当前组成绩到大屏 |

### 5.6 参数设置

点击右上角 **"⚙ 参数设置"** 按钮打开设置对话框：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 泳道关闭时间 | 20秒 | 发令后到触板打开的倒计时 |
| 出发台关闭延迟 | 3秒 | 出发后出发台延迟关闭时间 |
| 成绩确认关闭延迟 | 3秒 | 完赛后设备延迟关闭时间 |
| 抢跳判定阈值 | 0.10秒 | 小于此值判定为抢跳 |
| 分段成绩显示时长 | 5秒 | 分段成绩、名次、反应时的显示持续时间 |
| 第1名成绩停留时间 | 3秒 | 第1名成绩在滚动时间位置的绿色显示持续时间 |
| 终点位置 | 左端 | 终点（触板端）在泳池的哪一端 |

### 5.7 泳道实时状态说明

每道显示信息：

```
[道次] [左发令指示] [左设备状态] [姓名+进度条] [右设备状态] [右发令指示] [反应时 | 成绩 | 名次 | 备注]
```

- **设备状态圆点**: 绿色=打开, 灰色=关闭, 红色=损坏, 黄色=抢跳
- **进度条**: 显示方向箭头 + 倒计时；完赛后显示最终成绩
- **T按钮**: 手动触板按钮（左端/右端各一个）

### 5.8 方向/进度条显示格式

| 状态 | 显示 |
|------|------|
| 泳道关闭中（倒计时） | `▶ (15.5s) ▶▶▶` （第1箭头 + 时间 + 进度箭头） |
| 比赛中（触板已开） | `▶▶▶▶▶▶▶▶` （满箭头） |
| 完赛 | `== 54.60 ==` （最终成绩，金色） |
| DNS/DNF/DSQ | 状态文字 |

## 6. 成绩与排名

### 6.1 查看成绩

在 **"成绩与排名"** 标签页中：
1. 选择性别、项目、赛次
2. 成绩列表自动显示

### 6.2 晋级操作

### 6.3 成绩发布

点击"成绩发布"按钮，成绩将推送到连接的大屏显示器。

## 7. 纪录管理

在 **"赛事管理与报名"** → **"纪录管理"** 子标签中管理各类纪录。

### 7.1 纪录类型

系统支持以下纪录类型：
- 世界纪录、奥运会纪录、亚洲纪录、亚洲青年纪录
- 全国纪录、省运会纪录、赛会纪录

### 7.2 导入纪录

**方式一：一键导入世界纪录**

点击 **"导入标准纪录"** 按钮，自动导入内置的 42 条最新世界纪录（截至2025年）。

**方式二：下载模板填写后导入**

1. 点击 **"下载纪录模板"** 按钮，保存 CSV 文件
2. 用 Excel 打开 CSV 文件
3. 在空行中填写各项纪录（保持者、代表队、成绩、日期、地点）
4. 成绩格式：
   - 秒.百分秒，如 `20.91`
   - 分:秒.百分秒，如 `1:42.00`
   - 时:分:秒.百分秒，如 `14:30.67`
5. 保存 CSV 文件（建议选择"CSV UTF-8"格式）
6. 点击 **"导入CSV纪录"** 按钮选择文件导入

> Excel 保存的 CSV 默认使用系统编码（GBK），程序会自动识别编码，中文不会乱码。

### 7.3 查询纪录

使用顶部的筛选控件查询：
- **性别**: 全部/男/女/混合
- **项目**: 下拉选择
- **类型**: 世界纪录/奥运会纪录/... 等
- **关键词**: 模糊搜索保持者、代表队、地点

### 7.4 WR/CR 显示

比赛控制界面顶部自动显示当前项目的：
- **WR**: 世界纪录成绩
- **CR**: 赛会纪录成绩

## 11. EXE远程控制台

### 11.1 适用场景

当主服务器不便直接操作时（如主服务器在机房），可在裁判台电脑运行远程控制台，通过网络远程控制整个比赛。

### 11.2 界面说明

远程控制台的界面与主服务器的"比赛控制"界面**完全一致**：

- **左栏**: 赛程导航（上60%）+ 计时源对比（下40%）
- **中间**: 滚动时间 + 泳道实时状态（最多10道）+ 日志
- **右栏**: 比赛控制按钮 + 泳道操作 + 显示控制

### 11.3 连接服务器

1. 在顶部输入主服务器IP和端口：`192.168.1.100:3002`
2. 点击"连接"
3. 绿色指示灯亮起表示连接成功

### 11.4 独立操作

远程控制台可以**完全独立操作**整场比赛：
- 选择项目/组次
- 就位 → 发令 → 计时复位 → 确认成绩
- 标记 DNS/DNF/DSQ
- 手动输入成绩
- 全部打开/关闭泳道
- 参数设置
- 大屏显示切换

### 11.5 快捷键

| 快捷键 | 功能 |
|--------|------|
| F5 | 就位 |
| F6 | 发令 |
| F7 | 计时复位 |
| F8 | 确认成绩 |
| Ctrl+← | 上一组 |
| Ctrl+→ | 下一组 |

## 13. 网页检录台

### 13.1 适用场景

比赛开场前，裁判在检录处核对每组运动员是否到场（核对身份证件），并将未到场 / 弃权 / 犯规等情况记录到系统中。检录完成后打印一份纸质检录表供现场签字留档。

### 13.2 访问地址

```
http://主服务器IP:3002/checkin.html
```

首次打开需在右上角点击 **"服务器"** 按钮，输入主服务器 WebSocket 地址（默认 `ws://<当前host>:3002`）。设置会保存到浏览器本地。

### 13.3 界面布局

- **左侧 赛程导航**：与比赛控制一致的单元 / 项目 / 组次树，点击某组即可打开该组的检录表；已完成检录的组显示绿色 ✓。
- **右侧 检录表**：当前所选组次的运动员名单，列包括：
  - 泳道 / 号码布 / 姓名 / **身份证号** / 单位 / 备注（检录）
- **接力项目**：每支队自动展开为"队行 + 每一棒队员"，每位队员单独一行可独立勾选状态（号码布、姓名、身份证号、备注）。

### 13.4 操作流程

1. 从左侧赛程导航选择一个尚未开赛的组。
2. 对照身份证件核对每位运动员。
3. 对于未到场者，在 **备注** 列下拉选择 `DNS 弃权/未到`；特殊情况可选 `DNF`/`DSQ`。
4. 修改后底部统计实时更新"实到 / 总数 / 缺席队员"等。
5. 点击 **确认检录** 按钮，检录结果提交回主服务器；主服务器在该组开赛前即可知道谁不到场，自动把 DNS 运动员泳道跳过。
6. 点击 **打印** 按钮出具纸质检录表（A4 纵向），底部带检录裁判签字行。

### 13.5 打印抬头

打印页面顶部自动生成：
- 第一行：**赛事名称**（18pt 粗体，取自主服务器"赛事名称"字段）
- 第二行：性别 + 项目 + 赛次 + 第 N 组 + **"检录表"**
- 第三行：共 N 人/队

底部右下角预留 **检录裁判签字**行。

### 13.6 关闭浏览器页眉/页脚（重要）

浏览器在打印时默认会在**每一页的顶部和底部**加上自己的页眉页脚（例如"网页标题 / 时间 / URL / 页码"），会盖到检录表抬头上方。为获得最整洁的打印效果，请在打印对话框中**关闭浏览器的页眉页脚**：

| 浏览器 | 操作路径 |
|--------|----------|
| **Chrome / Edge** | 打印对话框 → 左侧"**更多设置**"展开 → **"页眉和页脚"** 复选框**取消勾选** |
| **Firefox** | 打印对话框 → **"更多设置"** → **页眉和页脚** 全部选 "无" / "--空白--" |
| **Safari** | 打印对话框 → **"显示详细信息"** → 取消勾选 **"打印页眉和页脚"** |

系统已在 JS 中临时把网页标题替换为赛事名称（即便忘记关闭页眉，页眉上也不会打印"游泳竞赛检录台"字样），但仍建议按上表彻底关闭以获得最干净的版面。

建议同时设置：
- **布局**：纵向 (Portrait)
- **纸张**：A4
- **边距**：默认 / 中等
- **缩放**：默认（100%）
- **每页一组**：单组运动员不多时默认即可；大组可在打印设置里将缩放改为 "适合页面宽度"。

### 13.7 接力项目的检录要点

- 每一棒队员都需单独核对身份证件并勾选
- 若某位队员缺席，可单独标记 DNS；教练可安排替补（通过主服务器"接力队管理"更换棒次）
- 若整队弃权，直接在**队行**（该队的第一行）将备注改为 DNS，所有后续成绩都会被跳过

## 15. 常见问题

### Q1: 远程控制台无法连接主服务器

**检查清单**:
1. 确认主服务器已启动
2. 确认两台电脑在同一局域网
3. 确认IP地址和端口正确（默认3002）
4. 检查防火墙是否放行了3002端口

### Q2: Web页面打不开

**解决方法**:
1. 确认主服务器已启动
2. 在主服务器电脑上先测试 `http://localhost:3002/race_control.html`
3. 如果本地可以打开但远程不行，检查防火墙设置

### Q3: 导入CSV纪录后中文乱码

**原因**: Excel 默认保存为 GBK 编码
**解决**: 程序已自动检测编码，通常不会出现此问题。如仍有乱码：
1. 在 Excel 中"另存为" → 选择 **"CSV UTF-8（逗号分隔）"** 格式
2. 重新导入

### Q4: 计时复位后 DSQ/DNS 状态消失了

**说明**: 当前版本已修复此问题。计时复位时 DSQ/DNS/DNF 等特殊状态会**自动保留**，只清除成绩数据。

### Q5: 分段成绩偶尔不显示

**说明**: 当前版本已修复。系统会跳过预创建的空分段，只显示实际触板返回的有效成绩。

### Q6: 第1名成绩没有在滚动时间位置显示

**说明**: 当前版本已修复。第1名成绩检测基于"最先收到的触板信号"，不依赖运动员匹配逻辑。

### Q7: 三个控制平台的时间不一致

**说明**: 当前版本已修复。所有客户端每次收到服务器广播（500ms一次）都会重新同步本地时钟，最大误差不超过网络延迟。

### Q8: 比赛中不小心点了其他项目

**说明**: 系统已加入保护机制。"就位"和"比赛中"状态下，禁止切换项目/组次。必须先点击"计时复位"才能切换。

### Q9: 网页检录表打印时页眉出现网址或"游泳竞赛检录台"字样

**原因**: 这是浏览器自动加的打印页眉（HTML 标准功能，CSS 无法控制）。

**解决**: 在浏览器**打印对话框**中关闭"页眉和页脚"：
- **Chrome / Edge**：打印对话框左栏 → "**更多设置**" → 取消勾选 "**页眉和页脚**"
- **Firefox**：打印对话框 → "更多设置" → **页眉和页脚**各项选 "**无**" 或 "**--空白--**"
- **Safari**：打印对话框 → "**显示详细信息**" → 取消勾选 "**打印页眉和页脚**"

系统已做了兼容处理（打印时临时把网页标题改为赛事名），即便忘了关闭也不会把"游泳竞赛检录台"打印在页眉上。详见第 13.6 节。
