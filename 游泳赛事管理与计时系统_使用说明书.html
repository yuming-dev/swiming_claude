一、系统概述

1.1 功能简介

本系统是一套完整的游泳比赛管理与计时解决方案，覆盖赛事全生命周期：

  • 赛前：赛事创建、运动员报名、自动生成日程、预赛分组编排、纪录数据管理
  • 赛中：实时计时控制、泳道设备管理、分段成绩显示、第1名成绩高亮、WR/CR纪录对比
  • 赛后：成绩确认与排名、晋级管理、成绩发布、文档打印（秩序册/出发表/成绩册等）

1.2 系统组成

组件说明运行方式
主服务器 (SwimmingScoreboard.exe)赛事管理核心，连接计时硬件Windows 应用程序
远程控制台 (RemoteTimingControl.exe)独立的比赛控制客户端Windows 应用程序
Web控制台 (race_control.html)浏览器端比赛控制界面任意浏览器打开
大屏显示 (display.html)观众大屏幕显示浏览器全屏
排名屏 (leaderboard.html)实时排名显示浏览器全屏
在线报名 (register.html)运动员自助报名浏览器打开

1.3 系统要求

  • 操作系统：Windows 7/8/10/11
  • 运行环境：.NET Framework 4.0（Windows 7+ 自带）
  • 网络：局域网（主服务器与客户端/大屏需在同一网络）
  • 端口：3002（WebSocket通信）
  • 浏览器：Chrome / Edge（推荐）

二、安装与启动

2.1 安装步骤

  1. 解压安装包 游泳赛事管理系统_安装包.zip
  2. 双击 安装.bat
  3. 输入安装目录（直接回车使用默认路径 C:\SwimmingTimingSystem）
  4. 安装完成后桌面出现两个快捷方式

2.2 启动主服务器

双击桌面 "游泳赛事管理主服务器" 快捷方式。首次启动需设置管理员密码，登录后进入主界面。

2.3 启动远程控制台

双击桌面 "远程计时控制台" 快捷方式。在顶部输入主服务器IP和端口（如 192.168.1.100:3002），点击"连接"。

2.4 打开Web控制台

在浏览器地址栏输入：http://主服务器IP:3002/race_control.html

2.5 防火墙设置

首次运行主服务器时，Windows 可能弹出防火墙提示。请选择 "允许访问"，确保端口 3002 可用。

四、赛事管理与报名

主服务器的 "赛事管理与报名" 标签页包含以下子标签：

4.1 赛事概览

显示当前赛事基本信息（名称、日期、地点等），统计已注册运动员数、项目数、团队数等概览数据。可编辑赛事名称、举办单位、裁判信息等。

4.2 运动员管理

  • 添加运动员：填写姓名、性别、代表队、出生日期、身份证号等
  • 导入CSV：批量导入运动员数据（支持 UTF-8 和 GBK 编码自动识别）
  • 参赛项目管理：为每位运动员分配参赛项目和报名成绩
  • 在线报名：运动员可通过 http://主服务器IP:3002/register.html 自助报名

4.3 接力队管理

创建接力队（4人一队），指定棒次顺序，设置接力项目。

4.4 出场编排微调

在自动分组后，手动微调运动员的组次和泳道。

4.5 赛程管理

  • 一键生成日程：根据报名数据自动生成完整赛程
    • 自动安排预赛 → 半决赛 → 决赛的时序（严格保证时间顺序）
    • 根据项目距离计算合理的时间间隔
    • 三阶段项目的预赛安排在前几天，保证半决赛和决赛有充足间隔
  • 预赛自动分组：按报名成绩蛇形分配泳道
  • 修改赛程：可手动调整日期、时间、组次

4.6 纪录管理

详见 第七章 纪录管理。

五、比赛控制

5.1 界面布局

区域内容
左栏赛程导航树（上60%）+ 计时源对比（下40%）
中间WR/CR纪录 + 滚动时间 + 泳道实时状态（10道）+ 日志
右栏比赛控制按钮 + 泳道操作 + 显示控制

5.2 比赛流程

第一步：选择项目

在左栏赛程导航树中点击要比赛的组次（如"女 100米蝶泳 预赛 第1组"）。

注意：比赛进行中（就位/比赛中状态）不能切换项目，必须先"计时复位"。

第二步：就位

点击 "就位" 按钮：状态变为"就位"，泳道设备重置为初始状态，出发端出发台打开。

第三步：发令

点击 "发令" 按钮：状态变为"比赛中"，计时器开始，所有触板和盲表关闭，泳道关闭倒计时开始。倒计时结束后触板/盲表自动打开。

第四步：比赛进行中

比赛中系统自动处理：

  • 泳道关闭/打开：按设定的倒计时自动控制
  • 分段成绩：每次触板触碰显示当前分段累计时间，保持设定秒数后消隐
  • 实时名次：与分段成绩同步显示/消隐
  • 反应时：出发台检测到反应时间后显示相同时长
  • 第1名成绩：每个分段点最先触板的成绩以绿色显示在滚动时间位置

第五步：确认成绩

所有泳道完赛后，点击 "确认本组成绩"，弹出确认对话框，确认后成绩锁定保存并自动检查纪录。

5.3 比赛控制按钮

按钮功能

5.4 泳道操作

需要先在输入框中输入道次号码，再点击操作按钮：

全部关闭关闭所有泳道设备触板和盲表全部关闭
说明：DNS/DNF/DSQ 操作都有确认对话框防止误操作。手动输入有二次确认后直接写入数据库。计时复位时 DSQ/DNS/DNF 状态不会被清除。

  • T按钮：手动触板按钮（左端/右端各一个）

六、成绩与排名

6.1 查看成绩

在 "成绩与排名" 标签页中选择性别、项目、赛次，成绩列表自动显示。

6.2 晋级操作

成绩确认后，可进行晋级操作：

  • 预赛 → 半决赛：取前16名（短距离项目）
  • 半决赛 → 决赛：取前8名
  • 预赛 → 决赛：取前8名（无半决赛时）

七、纪录管理

7.1 纪录类型

系统支持：世界纪录、奥运会纪录、亚洲纪录、亚洲青年纪录、全国纪录、省运会纪录、赛会纪录。

7.2 导入纪录

方式一：一键导入 — 点击"导入标准纪录"按钮，自动导入内置的42条世界纪录。

方式二：模板填写 — 点击"下载纪录模板"→ Excel打开填写 → "导入CSV纪录"读入。

成绩格式：20.91（秒）、1:42.00（分:秒）、14:30.67（分:秒）

7.3 查询纪录

使用顶部筛选控件（性别/项目/类型/关键词）查询。

7.4 WR/CR 显示

比赛控制界面顶部自动显示当前项目的 WR（世界纪录）和 CR（赛会纪录）。

十四、常见问题

Q1: 远程控制台无法连接主服务器

检查：①主服务器已启动 ②同一局域网 ③IP和端口正确 ④防火墙放行3002端口

Q2: 导入CSV纪录后中文乱码

程序已自动检测编码。如仍有乱码，在Excel中"另存为"选择 "CSV UTF-8" 格式后重新导入。

Q3: 计时复位后 DSQ/DNS 状态消失

当前版本已修复，计时复位时特殊状态自动保留。

Q4: 第1名成绩没有在滚动时间位置显示

当前版本已修复，基于"最先收到的触板信号"检测第1名。

Q5: 三个控制平台的时间不一致

当前版本已修复，客户端每500ms与服务器重新同步时钟。

Q6: 比赛中不小心点了其他项目

"就位"和"比赛中"状态下禁止切换项目，必须先"计时复位"。
